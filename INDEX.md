## 💾 快速命令参考

### 环境设置
```bash
# 安装依赖
pip install -r requirements.txt

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate
```

### 快速运行
```bash
# 一键快速开始
python3 quick_start.py

# 生成数据
python3 data_generator.py

# 下载真实数据
python3 tianchi_data_downloader.py
```

### Python代码示例
```python
# 加载数据
from data_loader import FinancialDataLoader
loader = FinancialDataLoader()
data = loader.load_alipay()

# 预处理
from data_loader import DataPreprocessor
preprocessor = DataPreprocessor()
processed = preprocessor.pipeline(data)

# 分割
from data_loader import DataSplitter
splitter = DataSplitter()
train, test = splitter.train_test_split(processed)
```
