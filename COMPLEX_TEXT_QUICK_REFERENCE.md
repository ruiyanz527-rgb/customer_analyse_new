## 🎯 新增字段

### 1️⃣ `kyc_complex_text`
**类型**：字符串  
**长度**：600-800字符  
**内容**：综合KYC分析材料

**包含内容**：
- ✅ 个人基本信息
- ✅ 工作背景（公司、职位、成就、薪资）
- ✅ 财务状况（收入、支出、交易、资产）
- ✅ 信用评分（芝麻分、借贷历史、评级）
- ✅ 风险评估（高/低风险指标）
- ✅ 行业分析（行业背景、薪资对比、趋势）

### 2️⃣ `reasoning_chain_detailed`
**类型**：字符串  
**长度**：400-600字符  
**内容**：5-6步推理链路

**推理步骤**：
```
[职业风险评估] ... | [收入匹配] ... | [信用评估] ... | 
[交易评估] ... | [综合评分] ... | [决策] ...
```

### 低风险用户示例

**KYC材料** (kyc_complex_text)：
```
【个人信息】
申请人：用户1234
年龄：35岁
学历：本科
城市：北京

【工作背景】
申请人在阿里巴巴担任产品经理一职6年，主要负责产品规划。
期间获得完成核心项目，建立了体系化的工作方法。
在公司期间累计薪资水平达到¥30K-¥50K，获得过卓越绩效奖等荣誉。
离职原因为寻求更好的发展机会。

【财务状况】
申请人年收入约¥300,000元，月均支出¥12,000元。近6个月的交易
记录显示，月均交易40笔，主要交易类型包括工资入账、日常消费、
转账、投资理财。申请人基金定投，目前资产充足。房产自有住房，
车产有车产。年收入稳定性为稳定，近3年收入趋势上升。无负债。

【信用评分】
芝麻信用评分为740分，在82%的用户中处于较高位置。申请人信用
记录良好，无逾期记录。申请人6个账户使用良好，查询适中。有借贷
记录，均按时还款。总体信用评级为优秀。正面因素：有稳定的收入
来源；负面因素：近期查询较频繁。综合评估，信用状况良好，可信度高。

【风险评估】
风险识别: 交易行为稳定。申请人消费模式合理，资金流向清晰。
基于交易模式分析，交易模式稳定，符合该群体的一般规律。
综合风险评分为0.18，风险等级为低风险。风险控制建议：可以进行
标准授信程序。

【行业分析】
行业背景：申请人所在互联网行业，发展前景广阔，薪资水平持续上升。
该行业平均薪资为¥200,000，申请人的收入水平高于行业平均。
近3年该行业保持增长，对申请人的职业发展有一定影响。
该职位发展潜力大，薪资增长空间明显。
```

**推理链路** (reasoning_chain_detailed)：
```
[职业风险评估] 申请人从事互联网行业，行业风险较低(10%)，职业发展潜力大|
[收入匹配] 申请人收入与年龄基本匹配，符合预期|
[信用优势] 芝麻信用分740分，高于平均水平，说明历史信用记录良好|
[交易正常] 月均交易40笔，交易活跃度正常|
[综合评分] 综合评分0.18，风险指标在可控范围内，属于低风险用户|
[决策] 建议可以进行标准授信，建议授信额度为¥1,500,000-¥2,500,000
```

### 高风险用户示例

**KYC材料** (kyc_complex_text)：
```
【个人信息】
申请人：用户5678
年龄：48岁
学历：高中
城市：深圳

【工作背景】
申请人在自营企业从事业务经理一职4年，主要负责团队管理。
期间获得带领团队，建立了流程优化的工作方法。在公司期间
累计薪资水平达到¥15K-¥25K，获得过年度最佳员工等荣誉。
离职原因为创业需要。

【财务状况】
申请人年收入约¥60,000元，月均支出¥3,500元。近6个月的交易
记录显示，月均交易68笔，主要交易类型包括工资入账、日常消费、
转账、投资理财。申请人股票投资，目前资产不足。房产租住，
车产无车产。年收入稳定性为波动，近3年收入趋势下降。负债较多。

【信用评分】
芝麻信用评分为480分，在35%的用户中处于较高位置。申请人有少量
逾期记录，但均已结清。申请人4个账户使用良好，查询频繁。无借贷
记录，均按时还款。总体信用评级为一般。正面因素：有多渠道的资产；
负面因素：负债率略高。综合评估，信用状况一般，需要关注。

【风险评估】
风险识别: 交易活跃度异常高。申请人频繁的大额转账，消费模式不稳定。
基于交易模式分析，交易模式显示存在异常资金流转，需要进一步核实。
综合风险评分为0.72，风险等级为高风险。风险控制建议：需要严格的额度
限制和交易监控。

【行业分析】
行业背景：申请人所在自营行业，人才需求旺盛，竞争压力较大。
该行业平均薪资为¥120,000，申请人的收入水平远低于行业平均。
近3年该行业有所下降，对申请人的职业发展有一定影响。
该职位专业性强，需要丰富的行业经验。
```

**推理链路** (reasoning_chain_detailed)：
```
[职业风险评估] 申请人从事自营行业，行业风险基数较高(30%)，需要重点关注收入稳定性|
[收入警告] 申请人年龄48岁但收入仅¥60,000，低于同龄平均水平，需要调查特殊原因|
[信用风险] 芝麻信用分480分，低于平均水平，需要谨慎评估|
[交易活跃] 月均交易68笔，经济活跃度异常高，可能存在套现|
[综合评分] 综合评分0.72，风险指标超过阈值，属于高风险用户|
[决策] 建议拒绝或严格控制额度，需要进一步的尽职调查
```

## 📞 常见问题

**Q: 文本太长会不会影响训练？**  
A: 使用梯度检查点和混合精度训练可以优化内存占用。建议使用支持长序列的模型如GLM-4。

**Q: 如何处理超长文本？**  
A: 使用滑动窗口或文本分段技术。参考GSPO训练指南中的长文本处理部分。

**Q: 推理链路顺序能改吗？**  
A: 推荐保持当前顺序（职业→收入→信用→交易→综合→决策），这样更符合金融风控逻辑。

**Q: 可以用于其他任务吗？**  
A: 可以。文本内容具有通用性，可适配其他长文本理解、多步推理任务。
